About Kraftfeld

Kraftfeld is a generative MIDI drum sequencer for iPad, designed for live EDM performers. It takes a simple 16-step pattern and evolves it over time with intelligent fills, phasing, and LFOs — making one bar sound like an ever-changing 64-bar performance.

Key Features

  • Generative drum sequencing from a single 16-step loop
  • Intelligent randomized phasing, fills, and per-step LFOs
  • Performance-optimized: mute, solo, bass sidechaining
  • MIDI output to any synth, drum machine, or DAW
  • Designed for live performance with zero menu diving
